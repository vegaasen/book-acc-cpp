# Filenames and Conventions

Folders: 
	
	chapter<n>/out
	chapter<n>/out/<file>.out
	chapter<n>/<file>.cpp

Filenames:

	CPP:
	e<exercice-number>p<part-number>c<chapter-number>.cpp

	OUT:
	<same as above>.out
